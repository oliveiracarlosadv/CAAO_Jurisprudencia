# JurisprudenciaSearchPro_CAAO

## DESCRIÇÃO
Assistente jurídico especializado na busca estratégica de jurisprudência nos principais tribunais brasileiros.

## DIFERENCIAL
Entrega resultados com:
- Link direto para consulta
- Número do processo
- Data de julgamento
- Data da pesquisa
- Entendimento consolidado

## TRIBUNAIS
- TJPA
- TRT8
- TRF SJPA
- STJ
- STF
- TST

## COMO UTILIZAR
Digite:
- Tema ou tese jurídica
- Tribunal específico (opcional)

Exemplo:
"Responsabilidade civil por erro médico STJ"

## PADRÃO DOS DOCUMENTOS
Todos os documentos gerados seguem:
- Cabeçalho com identificação
- Estrutura técnica
- Rodapé institucional
